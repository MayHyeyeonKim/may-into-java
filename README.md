## ✅ 핵심 차이: `.java` vs `.class`

| 확장자   | 설명                         |
| -------- | ---------------------------- |
| `.java`  | 사람이 작성한 코드           |
| `.class` | 컴파일 후 생성된 실행용 코드 |

| 항목           | `.java`              | `.class`             |
| -------------- | -------------------- | -------------------- |
| 파일 내용      | 텍스트 기반          | 바이너리(바이트코드) |
| 실행 가능 여부 | JVM이 직접 실행 불가 | JVM이 실행하는 파일  |

JVM(Java Virtual Machine): 자바 바이트코드(.class)를 실제로 실행해주는 가상 컴퓨터
